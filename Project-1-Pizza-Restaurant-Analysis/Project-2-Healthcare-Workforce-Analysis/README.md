# Healthcare Workforce Mental Health Analysis

## Project Overview
This is a practice project showcasing data analysis capabilities using a simulated healthcare workforce mental health dataset. The project demonstrates my ability to handle sensitive data topics, perform comprehensive analysis, and deliver professional insights.

## Objectives
- Practice analyzing complex datasets with multiple variables
- Demonstrate statistical analysis skills
- Create meaningful visualizations for healthcare data
- Build professional reporting capabilities
- Showcase ability to work with HR/healthcare-type data

## Tools Used
- **Microsoft Excel**: Data processing, statistical analysis, pivot tables, and data validation
- **Microsoft PowerPoint**: Data visualization, charts, and professional presentation

## Project Files
- `Healthcare_Workforce_Mental_Health_Analysis.pptx` - Complete presentation with insights and visualizations
- `Project_portofolio_2_Healthcare_Workforce_Mental_Health_Dataset.xlsx` - Practice dataset and analysis

## Key Questions Explored
- What are the primary mental health indicators in the dataset?
- Which departments or roles show higher stress patterns?
- How does workload correlate with wellbeing scores?
- What factors influence work-life balance ratings?
- What patterns emerge from the mental health support data?

## Key Insights (Based on Practice Data)
- Correlation identified between extended work hours and stress levels
- Departments with higher patient loads showed elevated burnout indicators
- Access to mental health resources correlated with improved wellbeing scores
- Work-life balance varied significantly across different shift patterns
