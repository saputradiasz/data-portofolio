# Pizza Restaurant Performance Analysis

## Project Overview
This is a practice project demonstrating data analysis skills using a simulated pizza restaurant sales dataset. The project showcases my ability to process, analyze, and visualize business data to extract meaningful insights.

## Objectives
- Practice data cleaning and preparation techniques
- Analyze sales patterns and customer behavior
- Create data visualizations and reports
- Demonstrate Excel proficiency for business analysis
- Build professional presentation skills

## Tools Used
- **Microsoft Excel**: Data processing, pivot tables, formulas, and analysis
- **Microsoft PowerPoint**: Data visualization and professional presentation

## Project Files
- `Pizza_Restaurant_Performance_Analysis.pptx` - Complete presentation with insights
- `project_portofolio_1_Pizza_restaurant_performance.xlsx` - Practice dataset and analysis

## Key Questions Answered
- What is the Total Revenue from sales?
- How many Total Orders were received?
- What is the average revenue per order?
- How many pizzas sold per pizza category?
- What are the top 10 best-selling pizzas?
- What is the revenue trend over time?
- What is the revenue by pizza category and size?

## Key Insights (Based on Practice Data)
- Classic Category pizzas are the highest contributor to total revenue
- Large size pizzas account for nearly half of total revenue
- Peak months: January ($71,620), March ($71,301), November ($71,005)
- Lowest month: December ($61,058) - possible seasonal decline
